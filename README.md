# Método de interpolación

El método de interpolación es un método científico lógico que consiste en determinar
 cada una de las variables en las formas en las que se pueden reproducir y cómo 
afectan al resultado. Pero no sólo basándose en su relación estadística sino también 
en su causalidad. 

Esto constituye las reglas que se utilizan para llegar a una nueva 
conclusión, siempre de forma aproximada. 

Es decir, se considera todas las situaciones 
posibles y sus repercusiones y las interpolamos a la nueva situación por analogía o 
inducción.
